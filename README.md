# Sales Data Analysis Dashboard

## Overview

This repository contains an interactive **Sales Data Analysis Dashboard** designed to provide insights into sales performance across different dimensions such as geography, promotions, and time.

The **Overview Dashboard** (shown above) serves as the central hub, offering a high-level summary of key business metrics and trends.

---

## Key Features of Overview Dashboard

### Sales by City

* Visual map representation of sales distribution across cities.
* Helps identify high-performing and underperforming regions.

### Number of Orders

* Displays the total number of orders (**3510** in the current snapshot).
* Acts as a quick KPI indicator for overall business activity.

### Average Discount by Promotion Categories

* Breaks down discount usage across different campaigns:

  * Weekend Flash Sale
  * Clearance Sale
  * Summer Sale
  * New Year Special
  * Festive Diwali
* Useful for evaluating promotion effectiveness.

###  Profit vs Net Sales

* Scatter plot showing the relationship between profit and net sales.
* Highlights correlation and helps identify profitability patterns.

###  Sales Trends Over Time

* Time-series visualization of sales performance (2020–2024).
* Enables trend analysis, seasonality detection, and anomaly spotting.

---

## 🔗 Connected Dashboards

The Overview Dashboard is integrated with multiple detailed dashboards for deeper analysis:

### 1. Top / Bottom 5 Analysis

* Identifies:

  * Top 5 performing products/customers/regions
  * Bottom 5 underperformers
* Helps prioritize business decisions and improvements.

---

### 2. Comparison Dashboard (Sales / Profit / Quantity)

* Compares key metrics:

  * Sales
  * Profit
  * Quantity
* Enables side-by-side evaluation across categories, regions, or time periods.

---

### 3. Edit Interaction Dashboard

* Allows customization of visual interactions.
* Users can:

  * Control filtering behavior
  * Adjust cross-highlighting between visuals
* Enhances user-driven exploration.

---

### 4. Table Visual Dashboard

* Detailed tabular view of raw and aggregated data.
* Supports:

  * Drill-down analysis
  * Sorting and filtering
* Ideal for granular inspection and reporting.
